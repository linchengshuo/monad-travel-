# 安全约束

## 严禁写进代码的内容

- 私钥。
- API Key。
- 主网部署私钥。
- 明文身份证。
- 明文电话。
- 明文姓名。

## 当前项目规则

- 所有私钥只能从 `$env` 或 `.env` 读取。
- `.env` 已经写进 `.gitignore`。
- Monad 测试网链 ID 固定为 `10143`。
- 部署脚本会拒绝非 `monadTestnet` 网络。
- 用户资料只生成哈希。
- 房型和库存保存在链外。
- Supabase 配置只能放在 `.env.local` 或部署平台环境变量里。
- `service_role key` 禁止进入前端项目。

## 后续必须补强

- 链外库存需要加锁定时间。
- 支付成功后需要监听链上事件。
- 支付失败后需要释放链外库存。
- 二维码需要加入一次性随机数。
- 入住签名需要防重放。
- 用户资料如果需要商家读取，必须做端到端加密。
- 商家入驻生产版应使用钱包签名 + 后端验证，避免匿名提交被刷。
